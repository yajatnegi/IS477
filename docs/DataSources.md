# Data Sources Documentation

## 1. Zillow Home Value Index (ZHVI)
- **Source:** https://www.zillow.com/research/data/
- **Description:** Monthly county-level estimates of typical home values.
- **Format:** CSV
- **Use:** We aggregate monthly values into an annual median for analysis.

## 2. USDA ERS Median Household Income
- **Source:** https://www.ers.usda.gov/data-products/county-level-data-sets/
- **Description:** Annual county-level median household income.
- **Format:** CSV
- **Use:** Provides socioeconomic indicator to compare against ZHVI values.
